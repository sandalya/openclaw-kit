## 🇺🇦 УКРАЇНСЬКА МОВА - Правильна Вимова

### Типові Помилки (Російські Калькування)

| ❌ НЕПРАВИЛЬНО | ✅ ПРАВИЛЬНО | ПРИМІТКА |
|---|---|---|
| написатимеш | напишеш | буд. час без суфіксу -и- |
| щодня вживання | щоденного використання | прикметник + іменник |
| мене як ідеш | мене по ходу | "як" не підходить тут |
| дивись, це я | дивись, це я (OK) | але "дивися" для звертання |
| очевидно | очевидно (OK) | або "очевидний" |
| звертаюсь | звертаюсь (OK) | або "звращаюсь" (застаріле) |
| поправляння | виправлення / коригування | "поправляння" звучить дивно |
| по ходу | по ходу (OK) | або "по дорозі" |
| як ти розумієш | як ти розумієш (OK) | або "наскільки я розумію" |

### Граматика

**Будущий час:**
```
❌ писатимеш → ✅ напишеш
❌ robиш тебе → ✅ робитимеш тобі
```

**Дієслова руху:**
```
❌ ходи → ✅ йди
❌ идёшь → ✅ йдеш
```

**Прикметники:**
```
❌ хороший → ✅ гарний / добрий
❌ интересный → ✅ цікавий
❌ важный → ✅ важливий
```

**Звертання:**
```
✅ "Розкажіть мені"
✅ "Дивись на це"
✅ "Слухайте, будь ласка"
❌ "Посмотри"
```

## ⚙️ РОЗРОБКА & ПРОЦЕСИ

### Перед запуском bot.py
```bash
# ЗАВЖДИ перевіряй дублікати:
cd insilver-v2/
bash check-before-start.sh

# Якщо systemd Active → НЕ запускай ручне:
systemctl status insilver-bot
```

**🚨 КРИТИЧНО:** Дублікати bot.py = конфлікт токенів = Кіт зникає (~10-й раз!)
**Наступний раз:** Кардинальне рішення (lock files/docker/guardian), не патчі!

### 🚀 ТВОЇ SSH АЛІАСИ

**Основні команди:**
```bash
startbot     # Запустити InSilver бот (через systemd)
stopbot      # Зупинити InSilver бот  
reset-bot    # Перезапустити тільки InSilver консультант
reset-all    # Перезапустити ВСЕ (InSilver бот + Кіт OpenClaw)
statusbot    # Показати статус InSilver сервісу
fix-dupes    # 🚨 АВТОФІКС дублікатів bot.py (~10-й раз!)
```

**Моніторинг:**
```bash
status       # Показати всі bot.py/agent.py процеси
logs-bot     # Останні 50 рядків bot.log  
```

**Різниця:**
- `reset-bot` = перезапуск тільки InSilver консультанта
- `reset-all` = перезапуск консультанта + Кіта (повний ресет)

**Як запитати аліаси у Кіта:** 
_"покажи мені мої аліаси"_ або _"які у мене команди для бота?"_
