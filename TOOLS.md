# TOOLS.md - Шпаргалка

## 🇺🇦 УКРАЇНСЬКА МОВА - Правильна Вимова

### Типові Помилки (Російські Калькування)

| ❌ НЕПРАВИЛЬНО | ✅ ПРАВИЛЬНО | ПРИМІТКА |
|---|---|---|
| написатимеш | напишеш | буд. час без суфіксу -и- |
| щодня вживання | щоденного використання | прикметник + іменник |
| мене як ідеш | мене по ходу | "як" не підходить тут |
| дивись, це я | дивись, це я (OK) | але "дивися" для звертання |
| очевидно | очевидно (OK) | або "очевидний" |
| звертаюсь | звертаюсь (OK) | або "звращаюсь" (застаріле) |
| поправляння | виправлення / коригування | "поправляння" звучить дивно |
| по ходу | по ходу (OK) | або "по дорозі" |
| як ти розумієш | як ти розумієш (OK) | або "наскільки я розумію" |

### Щоденні Фрази (Консультант)

```
ПРИВІТ:
✅ "Ювелірна майстерня InSilver, вітаємо Вас!"
✅ "Добрий день, як справи?"
❌ "Привет!" (російське)
❌ "Добрый день" (російське)

СЛУХАННЯ:
✅ "Розкажіть, що вас цікавить"
✅ "Слухаю вас"
❌ "Говори что хочешь" (російське)

ВАРІАНТИ:
✅ "Два варіанти: А або Б"
✅ "Яке вам подобається більше?"
❌ "Выбери одно" (російське)

ЧЕСНА:
✅ "Не знаю, переконаюсь у майстра"
✅ "Не впевнений, маєте час зачекати?"
❌ "Не уверен" (російське)

ПОДЯКА:
✅ "Дякую за замовлення"
✅ "Спасибі за вибір"
❌ "Спасибо" (російське)

ПРОЩАННЯ:
✅ "Бережіть себе!"
✅ "До скорого!"
✅ "Дякую, бережіть себе!"
❌ "До встречи" (російське)
❌ "Пока" (російське)
```

### Граматика

**Будущий час:**
```
❌ писатимеш → ✅ напишеш
❌ robиш тебе → ✅ робитимеш тобі
```

**Дієслова руху:**
```
❌ ходи → ✅ йди
❌ идёшь → ✅ йдеш
```

**Прикметники:**
```
❌ хороший → ✅ гарний / добрий
❌ интересный → ✅ цікавий
❌ важный → ✅ важливий
```

**Звертання:**
```
✅ "Розкажіть мені"
✅ "Дивись на це"
✅ "Слухайте, будь ласка"
❌ "Посмотри"
```
