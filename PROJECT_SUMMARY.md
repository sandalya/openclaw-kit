# PROJECT SUMMARY - InSilver v3

## Головна мета
Telegram бот-консультант для ювелірної майстерні Влада + dev-агент (Кіт) для розробки.

## Ключові компоненти
- **Консультант бот:** Спілкування з клієнтами, каталог, замовлення
- **Dev-агент:** Розробка, дебаг, документація, тестування
- **База даних:** Каталог товарів, замовлення, навчальні дані

## Поточний статус
- ✅ v2 працює в production
- 🔄 v3 в розробці - Telegram integration issues
- 🎯 Фокус: Cost optimization + stability

## Критичні вимоги
- **Навчання через /admin** - Влад може вчити бота
- **Стабільність** - мінімум дублікатів процесів/конфліктів
