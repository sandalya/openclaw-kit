# Правила Консультацій з Claude

## 💰 Автоматичні консультації (≤$0.01)
- Короткі технічні питання
- Простий код review  
- Швидкі рішення

## 🤔 З дозволом Сашка (>$0.01)
- Довгі архітектурні питання
- Консультації з великим контекстом
- Складні design decisions

## 📊 Оцінка вартості консультації:
```
Токени = (питання + контекст) * 1.3
Вартість = (input_токени * $0.003 + output_токени * $0.015) / 1000
```

Якщо >$0.01 → спитати дозвіл у Сашка з оцінкою вартості.
