# ⚡ ENERGY CONSUMPTION ISSUE - FIXED

## 🔴 ПРОБЛЕМА (2026-03-25 19:20)

**Симптом:** Енергоспоживання Pi5 подвоїлось з 600-800mA до 1-1.5A

## 🔍 ДІАГНОСТИКА  

**Причина:** OpenClaw застряг в infinite loop при завантаженні Ollama extension

**Процеси-винуватці:**
- PID 164434: openclaw-gateway 126.7% CPU, 1.4GB RAM
- PID 164790: openclaw-gateway 133.3% CPU, 1.2GB RAM  

**Симптоми:**
- Load average: 1.90+ (норма ~0.1)
- CPU: постійно 100%+ 
- RAM: споживання +1.5GB

## ⚒️ ВИПРАВЛЕННЯ

1. **Force kill** проблемних процесів:
   ```bash
   sudo kill -9 164434 164790
   ```

2. **Disable OpenClaw автозапуск** (через системний баг):
   ```bash
   sudo systemctl disable openclaw
   ```

3. **Залишити старий працюючий процес** (PID 19880, Mar24):
   - 6.7% CPU (нормально)
   - 820MB RAM (прийнятно)
