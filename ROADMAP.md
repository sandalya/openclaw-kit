## Layer 3 — KNOWLEDGE (стратегія, майбутнє)

Досліджую, планую, документую. Не впроваджую без обговорення з Сашком.

### Дослідити: Голосові повідомлення

З `FUTURE_IDEAS.md` + `FUTURE_QUESTIONS.md`:
- OpenAI Whisper API — розпізнавання української мови
- Workflow: голос → текст → GPT-4 → текстова відповідь (або голосова через TTS)
- Вартість: Whisper $0.006/хвилину → при 20 повідомлень/день ~$0.5-1/місяць
- Висновок для Сашка: **варто** якщо клієнти активно шлють голосові

### Дослідити: Багатомовність

З `FUTURE_QUESTIONS.md`:
```python
language_prompts = {
    'ua': 'Ти консультант ювелірної майстерні...',
    'ru': 'Ты консультант...',  # питання: Сашко хоче?
    'en': 'You are a jewelry consultant...'
}
```
- GPT-4 нативно підтримує — технічно просто
- Нюанс: назви плетінь (Козацьке, Бісмарк) — потрібен переклад термінів
- Рішення: таблиця термінів у `catalog.py`

### Дослідити: Viber інтеграція

З `FUTURE_QUESTIONS.md`:
- Viber Bot API — схожий на Telegram, але менша аудиторія в UA
- Питання Сашку: є клієнти які просять Viber?
- Технічно: окремий adapter, спільна бізнес-логіка
- Оцінка: 1-2 тижні роботи

### Дослідити: Автоматичне оновлення бази знань

З `FUTURE_QUESTIONS.md`:
- Парсити нові пости Влада в Telegram → автододавання в `training.json`
- Аналіз `conversations.log` → виявлення питань без відповіді → gap analysis
- Версіонування `training.json` через git

### Відкладено (обговорити з Сашком)
- OLX інтеграція — автопублікація товарів
- Redis/SQLite кеш для пошуку фото (зараз не потрібно при 10-20 повідомлень/день)
- Веб-панель для Влада
- Статистика та аналітика
