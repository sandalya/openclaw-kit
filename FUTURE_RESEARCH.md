## 🎯 Пріоритетні дослідження

### Дослідити: Голосові повідомлення

З `FUTURE_IDEAS.md` + `FUTURE_QUESTIONS.md`:
- **OpenAI Whisper API** — розпізнавання української мови
- **Workflow:** голос → текст → GPT-4 → текстова відповідь (або голосова через TTS)
- **Вартість:** Whisper $0.006/хвилину → при 20 повідомлень/день ~$0.5-1/місяць
- **Висновок для Сашка:** **варто** якщо клієнти активно шлють голосові

**Технічні деталі:**
```python
# Додати в bot handlers
@bot.message_handler(content_types=['voice'])
def handle_voice(message):
    # Download voice → Whisper API → process as text
```

З `FUTURE_QUESTIONS.md`:
```python
language_prompts = {
```

## 📦 Відкладені інтеграції

- **Складність:** висока, потребує OLX API access

- **Управління каталогом** через браузер

- **Індексування** по плетіннях та типах
