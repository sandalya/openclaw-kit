# SSH АЛІАСИ для InSilver v3 - ДОВІДНИК

_Оновлено: 2026-03-23_

## 🚀 ОСНОВНІ КОМАНДИ БОТА

| Аліас | Що робить | Використання |
|-------|-----------|-------------|
| `startbot` | Запускає InSilver v3 через systemd | Коли бот зупинений |
| `stopbot` | Зупиняє InSilver v3 | Для планових зупинок |
| `statusbot` | Показує статус сервісу insilver-v3 | Перевірка чи працює |
| `reset-bot` | Перезапускає тільки InSilver v3 | При зависанні бота |

## 📊 МОНІТОРИНГ

| Аліас | Що робить | Використання |
|-------|-----------|-------------|
| `status` | Показує всі процеси main.py та agent.py | Швидка перевірка |
| `logs-bot` | Останні 50 рядків логу InSilver v3 | Читання помилок |
| `logs-live` | Живий лог InSilver v3 (Ctrl+C щоб вийти) | Моніторинг в реальному часі |

## 🔧 СПЕЦІАЛЬНІ КОМАНДИ

| Аліас | Що робить | Коли використовувати |
|-------|-----------|---------------------|
| `reset-all` | Перезапускає ВСЕ: бот + OpenClaw | При серйозних проблемах |
| `fix-dupes` | Виправляє дублікати процесів bot.py | Коли кіт зникає в Telegram |

## 💻 РОЗРОБНИЦЬКІ

| Аліас | Що робить | Використання |
|-------|-----------|-------------|
| `dev-check` | Показує DEV_CHECKPOINT.md | Швидкий огляд стану проекту |
| `cd-bot` | Переходить до папки insilver-v3 | Для роботи з файлами |

## 🐱 OpenClaw (Кіт)

| Аліас | Що робить | Використання |
|-------|-----------|-------------|
| `gws` | Статус openclaw-gateway | Перевірка кіта |
| `gwr` | Перезапуск openclaw-gateway | При проблемах з кітом |
| `gwl` | Логи openclaw-gateway | Читання логів кіта |

## 🤖 ПЕРЕМИКАННЯ МОДЕЛЕЙ

| Аліас | Що робить | Модель |
|-------|-----------|--------|
| `use-haiku` | Переключає на швидку модель | claude-haiku-4-5-20251001 |
| `use-sonnet` | Переключає на збалансовану | claude-sonnet-4-6 |
| `use-opus` | Переключає на потужну | claude-opus-4-6 |
| `which-model` | Показує поточну модель | - |

## 📝 ПРИКЛАДИ ВИКОРИСТАННЯ

```bash
# Перевірка стану
status
statusbot

# Перезапуск при проблемах
reset-bot

# Читання логів
logs-bot
logs-live

# Повний ресет при серйозних проблемах  
reset-all

# При зникненні кіта в Telegram
fix-dupes

# Переключення на швидку модель
use-haiku
```

## ⚠️ ВАЖЛИВО

- **Всі аліаси тепер працюють з insilver-v3** (не v2)
- **reset-all** перезапускає і бота, і кіта
- **fix-dupes** вирішує ~10-у повторювану проблему з дублікатами
- **Після ssh** аліаси активні одразу (через ~/.bash_aliases)
